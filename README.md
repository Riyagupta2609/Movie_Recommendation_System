# Movie_Recommendation_System
A Movie Recommendation System built using Machine Learning and Streamlit that suggests similar movies based on content similarity using CountVectorizer and Cosine Similarity. Integrated with TMDB API to display movie posters and provide an interactive user experience.

# 🎬 Movie Recommendation System

A content-based movie recommendation web application built using Python, Machine Learning, and Streamlit. The system recommends movies similar to a selected title by analyzing movie genres and overviews using Natural Language Processing techniques.

## Features

* Movie recommendations based on content similarity
* Interactive Streamlit web interface
* Movie poster integration using TMDB API
* Genre-based and overview-based recommendation engine
* Fast similarity search using Cosine Similarity
* User-friendly movie selection dashboard

## Tech Stack

* Python
* Pandas
* Scikit-Learn
* Streamlit
* TMDB API
* Pickle

## Machine Learning Workflow

1. Data Preprocessing and Cleaning
2. Genre ID to Genre Name Mapping
3. Feature Engineering using Movie Genres and Overview
4. Text Vectorization using CountVectorizer
5. Similarity Calculation using Cosine Similarity
6. Recommendation Generation

## Dataset

TMDB Top 10,000 Movies Dataset from Kaggle.
